# Task 1 - Number Game

## Description

This is a simple Number Guessing Game developed in Java. The program generates a random number between 1 and 100, and the user has to guess the correct number.

## Features

* Generates a random number between 1 and 100
* Takes user input
* Provides hints if the guess is too high or too low
* Continues until the correct number is guessed

## Technologies Used

* Java
* Scanner Class
* Random Class

## How to Run

Go To Numbergame.java

## Learning Outcomes

* Random Number Generation
* Loops in Java
* Conditional Statements
* User Input Handling
